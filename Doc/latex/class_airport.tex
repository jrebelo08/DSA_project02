\doxysubsection{Detailed Description}
Represents an airport entity with specific attributes. 

This class defines the properties of an airport such as its unique code, name, city, country, latitude, and longitude. The class provides methods to access and modify these properties. Additionally, it overloads certain operators for comparison purposes.
