Find optimal flight paths while avoiding specific airlines. 

This function identifies optimal flight paths between a set of possible source airports and a set of possible destination airports, with the capability to avoid specific airlines as specified by the user.

Find optimal flight paths while minimizing airline changes. 

This function identifies optimal flight paths between a set of possible source airports and a set of possible destination airports, with a focus on minimizing the number of different airlines used during the path.

Counts the minimal amount of \doxylink{class_airline}{Airline} changes per path in a given vector of minimal \doxylink{class_airport}{Airport} paths The final vector will only contain paths that do not surpass the calculated amount
